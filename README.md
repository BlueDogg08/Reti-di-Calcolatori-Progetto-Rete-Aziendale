# Reti-di-Calcolatori-Progetto-Rete-Aziendale
Progetto universitario del corso “Reti di Calcolatori: Protocolli” — Simulazione e documentazione di una rete aziendale realizzata con Cisco Packet Tracer. Include schema logico e fisico, configurazioni DNS, Mail, Web, Firewall, e documentazione completa.
